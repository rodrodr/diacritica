This folder is used to store local images for the website.

How to use:
1. Upload your images (e.g., my-photo.jpg) into this folder.
2. Open `public/data.json`.
3. Locate the item you want to update.
4. Change the "image" or "contentImage" field to point to your new file:
   
   "image": "images/my-photo.jpg"

Note: Ensure the file names match exactly.
